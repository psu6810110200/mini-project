web ขายอาวุธสงคราม

สมาชิกในกลุ่ม
1. [ปัณณวิชญ์ กังพานิช] (รหัสนักศึกษา: 6810110200)
2. [อิศม์อนีติ เพ็งแจ่ม] (รหัสนักศึกษา: 6810110432)

เกี่ยวกับโปรเจกต์
โปรเจกต์นี้เป็น Full Stack Web Application ที่พัฒนาขึ้นเพื่อจำลองระบบ E-commerce สำหรับการซขายอาวุธสงคราม โดยเน้นการออกแบบระบบที่มีความปลอดภัยและการจัดการข้อมูลที่มีความซับซ้อน

ฟีเจอร์หลัก
1. Authentication:* ระบบสมัครสมาชิกและเข้าสู่ระบบด้วย JWT (แยกสิทธิ์ Admin/User)
2. Product Management:* Admin สามารถ เพิ่ม/ลบ/แก้ไข ข้อมูลอาวุธและจัดการ Stock ได้
3. Order System:* User สามารถเลือกสินค้าลงตะกร้าและกดสั่งซื้อได้
4. Transaction:* ระบบตัด Stock สินค้าแบบ Real-time และบันทึกประวัติการสั่งซื้อ
